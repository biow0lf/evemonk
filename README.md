EveMonk is helper software for EveOnline MMORPG.

## Features

* API signup
* API session
* Multiple sessions for clients

## Environment

* Ruby 2.3.3
* RubyGems 2.6.8
* Bundler 1.13.6
* Ruby on Rails 5.0.0.1
* PostgreSQL 9.5.4
